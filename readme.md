Утилита для поиска наиболее для вас подходящих mtproto proxy серверов для telegram.
Вернее, это просто батник-обертка которая базируется на [telegram-proxy-collector](https://github.com/kort0881/telegram-proxy-collector) и упрощает запуск поиска прокси для обычного пользователя.

Данная утилита в первый запуск скачает и настроит все необходимое для работы, по этому не пугайтесь.
Последующие запуски поиска будут более быстрые и спокойные.

Использование:
1. Скачайте zip-архив: зеленая кнопка `<> Code` -> `Download ZIP`
2. Разархивируйте в любое место
3. Запустите от имени администратора файл `proxy_search` или `proxy_search.bat`
